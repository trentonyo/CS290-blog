Entry #3

11/4/2022
What was challenging about the assignment, and what specific kinds of problems did you have. How did you solve those problems?
Like I'm sure many others experienced, the filtering task was nontrivial. The actual manipulation of the DOM was not the challenge in my case, that bit was fairly simple (tokenize the posts, save a list, work off of that). It was the actual logic of the filter that I was struggling with a tiny bit - until I read the instructions a little more closely and found that I was over/underdoing it by having an inclusive filter instead of an exclusive one as was required.

What did you learn from the assignment? Were there any special insights you had? What did you find that you already knew?
Finally understanding anonymous functions is a real game-changer!

Is there anything extra you added beyond the scope of the assignment?
I added a little quality of life improvement to the filtering system, if the user presses 'Enter' in any of the text entry areas (text, min or max price), the filter will update as if they pressed the 'Update' button in the filter dialog. I did this by adding a keypress listener to those elements and calling the update filter function if the key pressed is 'Enter.'
