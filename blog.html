Entry #2

10/21/2022
What was challenging about the assignment, and what specific kinds of problems did you have. How did you solve those problems?
The most challenging thing for me was getting a good idea of the scale of the mockups, but this was only a minor issue! Viewing the mockups in my browser instead of using the lackluster Windows image viewer app made the goal of a 1-to-1 recreation easier.

What did you learn from the assignment? Were there any special insights you had? What did you find that you already knew?
I never looked too deeply into selectors and combinators when I was learning CSS back in the day, so I kind of went wild with them on this assignment to get some experience in!
Another new thing to me were flexboxes, what a handy tool they are. I will be using them a lot going forward!

What resources (e.g. specific web articles, the class Campuswire forum, the TAs) were most helpful in completing this assignment? How did you use these resources?
  • Classmates - A few ticky-tacky details were pointed out by some of my buddies in class, they helped me notice those details and correct them!
  • MDN Box Shadow Generator - A super handy tool that allowed me to graphically generate the drop shadow on the sell item button (+)

What are one or two things you had to Google to complete the assignment?
  • Cursor: I had to look up which cursor was the pointy-finger one, felt silly to find that it was called "pointer"
  • Centering Images: This is something I've never managed to remember, so I had to look up how to center an img again for about the ten millionth time.

Entry #1

9/27/2022
What challenges or troubles did you have completing this assignment. How did you work through them?
Consider my troubles a cautionary tale! I use WebStorm from JetBrains as my IDE, and it has GitHub integration; my first attempts to clone the assignment repository were met with an error saying that the repo "could not be found" when I attempted to do so through WebStorm. My solution was to clone using the SSH protocol, as suggested by a classmate.

What did you learn from this assignment? What did you already know?
I've always been a little gun-shy about using terminal commands, but it was suggested that I get over myself and use git commands - and I'm glad I did!
Having done some amount of amateur web development, I am familiar with HTML, CSS, and JS (also PHP, but don't tell anyone)

What resources (e.g. specific web articles, the class Campuswire forum, the TAs) were most helpful in completing this assignment? How did you use these resources?
  • Github Pages - This article was linked in our assignment README and taught me how to host my site for free on GitHub!
  • W3Schools - I often refer back to this site when I forget the syntax or how something behaves in HTML, CSS, or JS! (I only used it for CSS this time around, specifically)
  • Coolors - I'm not great with color palettes, so I like to cheat a little and use this tool (and it's still not very congruent)
  • CampusWire - As I mentioned above, one of our peers suggested that I use git commands and gave me some pointers about it.
